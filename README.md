# Feedback & Fragen
Sie können **Fragen stellen** oder **Kommentare hinterlassen**, indem Sie ein [neues Issue öffnen](https://github.com/WolfgangErbe/copilot-chat-klassenskaempfe/issues/new).

**Bitte geben Sie an:**
- Zu welchem **Kapitel/Abschnitt** Ihr Feedback gehört.
- Ihre **konkrete Frage oder Anmerkung**.
- Optional: **Kontext** (z. B. Zitate oder Links).

⚠️ **Hinweis:** Dieses Repository ist **nur lesbar**. Änderungen sind nicht möglich.
